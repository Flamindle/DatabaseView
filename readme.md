环境准备：
安装 Node.js（建议 v14+）
初始化项目：将上述文件放在同一目录下
安装依赖：执行 npm install
运行步骤：
启动后端服务：npm start
打开浏览器访问：http://localhost:3000
输入 MySQL 连接参数（主机、端口、用户名、密码、数据库名）
点击 "连接数据库"，成功后选择表并点击 "查询数据"
注意事项：
确保 MySQL 服务已启动并允许外部连接
数据库用户需要有对应的查询权限
该示例为演示用途，生产环境需添加参数加密、权限控制、SQL 注入防护等
四、总结

核心架构：采用 Node.js+Express 后端处理数据库连接，前端通过 AJAX 与后端交互，实现参数提交和数据展示
核心功能：支持 MySQL 连接参数配置、连接测试、表列表获取、表数据查询展示
关键技术：使用 mysql2 库处理 MySQL 连接（支持 Promise），Express 搭建 REST 接口，前端原生 JS 实现交互和数据渲染
该方案实现了基础的 MySQL 数据库连接和数据查询功能，代码结构清晰，适合新手学习和二次开发。如果需要扩展功能（如分页、条件查询、数据编辑等），可以在此基础上进行修改。# DatabaseView
